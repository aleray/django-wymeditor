Django wymeditor widget

This is simple widget for django forms

Example

{{ form.as_p }}

Installing

Use in django-admin

Just inherit your admin class from RichTextAdmin: from wymeditor.admin import RichTextAdmin 
        class YourModelAdmin(RichTextAdmin):
            pass

or as mixin: 
        class YourModelAdmin(YourCustomAdminClass, RichTextAdmin):
            pass

Use in forms

        from django import forms
        from wymeditor.widgets import WYMEditor

        class MyForm(forms.Form):
            my_field = forms.CharField(widget=WYMEditor())

Customization

WYMEditor take argument skin (default='django')
